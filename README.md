# LLM Research Bot

A tool to help researchers find relevant papers for literature reviews based on keywords.

## Features

- Search arXiv for papers matching your keywords
- Use topic modeling and clustering to identify the most relevant papers
- Generate summaries of selected papers
- Export results for easy citation

## Setup

1. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

2. Run the main application:
   ```
   python main.py
   ```

## Usage

1. Enter keywords related to your research
2. The tool will search for relevant papers
3. Papers will be clustered by similarity
4. The most relevant papers will be selected
5. Summaries will be generated for the selected papers
